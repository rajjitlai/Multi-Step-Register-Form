# Multi-Step Registration Form — Premium UI

A professional, high-end registration form featuring a modern **glassmorphic** design, smooth animations, and real-time progress tracking.

## ✨ Features

- **Premium Glassmorphism**: Stunning translucent UI with backdrop-blur and vibrant accents.
- **Progress Tracking**: Dynamic progress bar with minimalist iconography.
- **Smooth Navigation**: High-performance CSS transitions and JS scroll logic.
- **Bi-directional Flow**: Fully functional "Next" and "Back" navigation.
- **Responsive & Modern**: Built with HTML5, CSS3 (Flexbox), and Vanilla JavaScript using the **Inter** font family.
- **SEO Optimized**: Includes proper meta tags and semantic structure.

## 🚀 Getting Started

### Prerequisites

No special dependencies are required. Just a modern web browser.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rajjitlai/Multi-Step-Register-Form.git
   ```
2. Open `index.html` in your browser.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with CSS Variables and Flexbox.
- **JavaScript**: Clean, modular Vanilla JS logic.
- **Google Fonts**: Inter.
