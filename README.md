# Multi-Step Registration Form — Premium UI

A professional, high-end registration form featuring a modern **glassmorphic** design, smooth animations, and real-time validation.

## ✨ Features

- **Premium Glassmorphism**: Stunning translucent UI with backdrop-blur and vibrant Emerald accents.
- **Progress Tracking**: Dynamic progress bar with professional Font Awesome icons and smooth CSS transitions.
- **Smart Validation**: Integrated HTML5 and JavaScript validation to ensure all required fields are correctly filled before proceeding.
- **Bi-directional Flow**: Fully functional "Next" and "Back" navigation with high-performance JS scroll logic.
- **Responsive & Modern**: Built with HTML5, CSS3, and Vanilla JavaScript using the **Inter** font family.
- **SEO Optimized**: Includes proper meta tags, semantic structure, and accessible button labels.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure and native form validation.
- **CSS3**: Vanilla CSS with CSS Variables, Flexbox, and complex animations.
- **JavaScript**: Modular, event-driven Vanilla JS (no dependencies).
- **Icons & Fonts**: Font Awesome 6 and Google Fonts (Inter).

## 🎨 Customization

This project uses **CSS Variables** located in the `:root` of `style.css`. You can easily change the entire theme by updating these tokens:

```css
:root {
    --primary: #80ed99;       /* Main accent color */
    --bg-dark: #05070a;       /* Deep dark background */
    --card-bg: rgba(15, 23, 42, 0.6); /* Glassmorphic card color */
}
```
